    OMI. Open Modular Interpreter

    OMI is one interpreter that processed source code written in OMI language. OMI is academic software and he is used for learning about how interpreters work.

    Interpretere software can be run as server software. This receives source code by a TCP port.

    When OMI processed source code can produce a JSON struct that describes the syntatic and semantic process carried out. This data can be represented by one client software, allowing interaction with the interpreting process.

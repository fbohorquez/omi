    Características de PcCom Basic Enterprise i5-4460/ 4GB/ 1TB

    Excelente relación calidad/precio para este PC con componentes de primeras marcas, la máxima garantía y gran rendimiento, ideal para usuarios que buscan hacer "de todo un poco".

    Pensado para un uso general en el hogar o en la oficina, el PcCom Enterprise se presenta como un equipo equilibrado en el que su procesador i5, sus 4GB de RAM y el generoso disco duro de 1TB permiten trabajar de forma holgada en muy diferentes tareas, lo que lo hace ideal para entornos domésticos o de oficina.

    • Intel Core i5-4460 3.2Ghz 
    • Disco Duro 1TB
    • Memoria Ram 4GB DDR3
    • USB 3.0
    • Lector Tarjetas + SIM + DNIe + Bluetooth

    Especificaciones:

    • Caja Nox Sense 500W
    • Procesador Intel Core i5-4460 3.2Ghz 
    • Placa Base Gigabyte GA-H81M-S2H
    • Disco Duro 1TB Western/Seagate 
    • Memoria Kingston ValueRAM 4GB DDR3 1333 PC3-10600 CL9 (1x4GB, 1 slot libre)
    • Grabadora DVD Sata Negra
    • Tarjeta gráfica integrada
    • Tarjeta de sonido integrada
    • Tarjeta de red 10/100/1000 integrada
    • Puertos de almacenamiento 
      • 4 x SATA
    • Posibilidad de conectar tarjeta gráfica en el puerto libre PCI Express.
    • Conexiones delanteras:  2 x USB2.0, 1 x Micrófono, 1 x Altavoces/auricular frontal 
    • Conexiones traseras
      • 1 x puerto RJ-45
      • 2 x puerto USB 2.0/1.1
      • 4 x puerto USB 3.0/2.0
      • 1 x HDMI
      • 1 x DVI-D
      • 3 x jacks de audio (Line In, Line Out, Mic In)
      • 1 x puerto de ratón PS / 2
      • 1 x puerto de Teclado PS / 2
      • 1 x puerto D-Sub
    • Dimensiones:  175 x 352 x 380 mm
    • No incluye Sistema Operativo.
    • Sistema operativo compatible: Compatible con Windows Vista, Windows 7, Windows 8, etc. Se se puede instalar el S.O. de forma sencilla.
    • Equipo totalmente montado y testeado
    • Garantía completa con recogida y entrega en domicilio.

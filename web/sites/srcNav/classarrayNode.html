Descripción detallada

Nodo array.

Representa un vector de elementos. Los array son nodos que presenta una tabla de símbolos interna en la que almacena referencias a cada uno de los elementos en el array.

En la inicialización del árbol de ejecución el nodo array se asocia con una lista de nodos de listas (listNode).

En la ejecución del nodo array los valores de la lista son guardados en la tabla de símbolos del array. En el caso de que los valores sean nodos asociativos (

Ver también
mapNode) se establece la clave como referencia. En caso contrario se utiliza como referencia un entero correspondiente a la posición del elemento en la lista.

Todo nodo array tiene asociado un nodo clase "arrayClass".
